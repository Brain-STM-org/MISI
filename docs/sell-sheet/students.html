Machine Intelligence Senior Internship (MISI)

Build Real Software with AI in 4 Weeks

I'm Interested

What if you could build anything?

Software development has changed. With AI tools like Claude Code, you can go from idea to working app in hours, not months. The catch? You need to know how to direct the AI, think like an engineer, and build something people actually want.

That's what MISI teaches you—in 4 intensive weeks with a team of your peers and real software engineers as mentors.

What You'll Build

You and your team pick a real project. Not a tutorial. Not a toy. Something you actually care about.

Your Idea

Pitch a project you're passionate about, or join a team working on something that excites you.

Real Users

Ship software that people outside your team actually use and give feedback on.

Your Portfolio

Walk away with deployed code, a GitHub profile, and something real to show colleges or employers.

What You'll Learn

AI-Assisted Development

Use Claude Code and other AI tools to write, debug, and ship code faster than ever before.

Software Engineering

Version control, testing, code review, and the principles that separate good code from garbage.

Team Collaboration

Work in small teams, run daily standups, review each other's code, and ship together.

Product Thinking

Learn to build what users need, gather feedback, and iterate quickly.

The 4 Weeks

Week 1: Launch

Form teams, pick projects, learn the tools, and push your first code.

Week 2: Build

Rapid iteration to a working MVP. Add key features based on feedback.

Week 3: Expand

Complete core features, get feedback from other teams. Feature freeze by Friday.

Week 4: Ship

Polish, test, deploy your project, present on Demo Day, and reflect on what you built.

"Garbage In, Garbage Out is more relevant than ever. AI can write code fast—but only you can make it good."

Is This For You?

You're curious about tech

You don't need to know how to code (yet). You need to want to learn.

You want to build, not just learn

This isn't a lecture series. You'll be writing code from day one.

You work well with others

Solo geniuses need not apply. This is about teams shipping together.

Questions?

Do I need to know how to code?

No prior coding experience required. You'll learn alongside AI tools that help you write code. What matters is curiosity and willingness to learn.

How much time does it take?

The program runs for 4 weeks, 25+ hours per week, with daily in-person sessions. Expect to spend significant time on your project—this is an internship, not a club.

What do I get at the end?

A deployed project you can show off, experience working on a real team, skills in AI-assisted development, and a network of peers who build things.

Does this count for my senior project?

We're working with schools to ensure MISI can fulfill senior project or internship requirements. Talk to your counselor.
