Machine Intelligence Senior Internship (MISI)

About the Program

MISI is a 4-week internship program for graduating high school seniors to explore new models of software development. Software engineering is in an LLM-assisted renaissance, where ideas become working code faster than ever before.

Students work in teams of 2-4, guided by experienced software engineers, to build real software projects using AI-assisted development tools like Claude Code. The program combines inquiry-based learning with proven software engineering principles—because while AI can write code fast, only humans can make it good.

From team formation to deployed product in 4 weeks. No prior coding experience required—just curiosity and a willingness to build.
