# AGENTS: Machine Intelligence Senior Internship  (MISI)

## Project Overview

**MISI** , the Machine Intelligence Senior Internship, is a 5-week internship program to explore new models of software development.  As an AGENT, you will assist in defining the program structure as well as educational materials and program processes.  This folder holds these artifacts, which we are co-develpoing using LLMs.
