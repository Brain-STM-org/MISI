# MISI Program Schedule

4-week program structure for Greenwich High School Senior Internship.

**Requirements:**
- Minimum 25 hours/week
- Hours logged via SchoolLinks
- Co-located for significant portions
- Demo Day at conclusion

## Kick-off Weekend

**Purpose:** Welcome, set tone, assign prep work

| Activity | Duration | Notes |
|----------|----------|-------|
| Welcome & introductions | 1 hour | Meet cohort, mentors |
| Program overview | 30 min | Expectations, schedule, tools |
| Prep homework assigned | — | Module 00: Development Environment |

**Homework (before Day 1):**
- Install VS Code
- Open terminal, practice basic commands
- Complete Module 00 exercises
- ~30 minutes total

## Flexibility Notes

- Daily schedule can flex based on team needs
- Just-in-time sessions may move between days
- Intensive Week 1 frontloads learning
- Weeks 2-3 prioritize project work
- All hours logged via SchoolLinks
- Balance intensity with joyfulness
